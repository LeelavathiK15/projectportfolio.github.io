PROJECT DETAILS

PROJECT TITLE:

EXPLORATION ENCRYPTION SCHEME BASED ON HASHCHAIN USING MD5 ALOGRITHM

ROLE :

Team Lead

TECHNOLOGY:

Cloud Computing

DESCRIPTION:

 We introduce a secure and practical Searchable Symmetric Encryption (SSE) scheme with provable security
 strength for cloud applications, called IDCrypt, which improves the search efficiency and enhances the 
 security strength of SE using symmetric  skills cryptography. We further point out the main challenges in securely
searching on multiple indexes and sharing encrypted data between multiple users we propose a token-adjustment search 
scheme to preserve the search functionality among multi-indexes, and a key sharing scheme which combines 
Identity-Based Encryption (IBE) and Public-Key Encryption.

 MODULE DESCRIPTION
  1. CLOUD SERVER
  2. SEARCHABLE ENCRYPTION
  3. MULTILEVEL INTERRELATIONSHIP
  4. AES CRYPTOSYSTEM.
